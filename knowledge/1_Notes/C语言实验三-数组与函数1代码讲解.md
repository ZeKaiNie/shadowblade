## E3-1.C：查找行最大且列最小的元素

### 题目功能

输入一个 `3 × 3` 的整型数组，找出这样的元素：

- 它是所在行中最大的元素
- 它同时是所在列中最小的元素

例如输入：

```text
1 2 3
4 5 6
7 8 9
```

第一行最大值是 `3`，它在第 `0` 行第 `2` 列。第 `2` 列是 `3, 6, 9`，其中 `3` 最小，所以输出：

```text
row=0,column=2,value=3
```

### 程序思路

每一行都检查一次：

1. 先找当前行最大值 `max`
2. 记录最大值所在列号 `col`
3. 再检查 `max` 是否是该列最小值
4. 如果满足，就输出它的位置和值

### 变量含义

- `a[3][3]`：保存 `3 × 3` 的二维数组
- `i`：行下标
- `j`：列下标
- `k`：检查列时使用的循环变量
- `max`：当前行最大值
- `col`：当前行最大值所在列
- `flag`：标志变量，`1` 表示符合条件，`0` 表示不符合

### 关键代码解释

输入数组：

```c
for (i = 0; i < 3; i++)
{
    for (j = 0; j < 3; j++)
    {
        scanf("%d", &a[i][j]);
    }
}
```

二维数组用两层循环输入。外层循环控制行，内层循环控制列。

查找当前行最大值：

```c
max = a[i][0];
col = 0;
for (j = 1; j < 3; j++)
{
    if (a[i][j] > max)
    {
        max = a[i][j];
        col = j;
    }
}
```

先假设当前行第 `0` 列最大，然后从第 `1` 列开始比较。如果发现更大的数，就更新 `max`，同时更新列号 `col`。

检查是否为列最小：

```c
flag = 1;
for (k = 0; k < 3; k++)
{
    if (max > a[k][col])
    {
        flag = 0;
        break;
    }
}
```

先假设 `max` 是该列最小值。如果发现同一列中存在比 `max` 更小的数，说明条件不成立，把 `flag` 改成 `0`。

### 原题改错点

1. 数组下标错误：原题从 `1` 循环到 `3`，但 C 语言数组下标应为 `0, 1, 2`。
2. 列号记录错误：找到行最大值时，应写 `col = j`，不能写 `col = i`。
3. 检查列最小时，应比较 `a[k][col]`，表示同一列不同行的元素。
4. 判断条件应使用 `==`，不能使用 `=`。`if (flag == 1)` 是判断，`flag = 1` 是赋值。

## E3-2.C：十进制数转换为 k 进制

### 题目功能

输入一个十进制整数 `a` 和一个进制 `k`，其中 `2 <= k <= 9`，输出 `a` 的 `k` 进制表示。

例如输入：

```text
8 2
```

输出：

```text
1000
```

### 核心方法：除基取余，倒序输出

以 `8` 转二进制为例：

```text
8 / 2 = 4 余 0
4 / 2 = 2 余 0
2 / 2 = 1 余 0
1 / 2 = 0 余 1
```

余数得到顺序是 `0 0 0 1`，实际输出要倒过来，即 `1000`。

### 关键代码解释

```c
aa[i] = a % k;
a = a / k;
```

- `a % k` 得到当前最低位
- `a / k` 去掉已经处理过的最低位

倒序输出：

```c
for (; i; i--)
{
    printf("%d", aa[i - 1]);
}
```

因为余数先存低位，后存高位，所以要从最后一个下标开始输出。

### 原题填空点

三个空分别是：

```c
aa[i] = a % k;
a = a / k;
printf("%d", aa[i - 1]);
```

## E3-3.C：统计字符类型个数

### 题目功能

输入不超过 `50` 个字符，统计其中：

- 英文字母个数
- 空格字符个数
- 数字字符个数
- 其他字符个数

### 为什么字符数组是 `str[51]`

题目说最多输入 `50` 个字符，但 C 语言字符串末尾还需要一个结束符 `\0`，所以数组至少需要 `51` 个元素。

### 为什么用 `fgets`

如果使用：

```c
scanf("%s", str);
```

遇到空格就会停止输入，不能统计空格。

所以使用：

```c
fgets(str, 51, stdin);
```

它可以读取包含空格的一整行。

### 分类判断

判断英文字符：

```c
(str[i] >= 'a' && str[i] <= 'z') || (str[i] >= 'A' && str[i] <= 'Z')
```

判断空格：

```c
str[i] == ' '
```

判断数字字符：

```c
str[i] >= '0' && str[i] <= '9'
```

其他情况归为其他字符。

### 容易被问到的问题

为什么循环条件要排除换行符？

```c
str[i] != '\0' && str[i] != '\n'
```

因为 `fgets` 可能把回车换行也读入字符串，如果不排除 `\n`，它可能被统计为其他字符。

## E3-4.C：判断两个字符串是否相等

### 题目功能

输入两个字符串，判断它们是否完全相等。

例如输入：

```text
abc abc
```

输出相等。输入：

```text
abc abd
```

输出不相等。

### 变量含义

- `str1[81]`：第一个字符串
- `str2[81]`：第二个字符串
- `j`：当前比较的字符下标
- `flag`：标志变量，`1` 表示相等，`0` 表示不相等

### 字符串比较逻辑

```c
while (str1[j] == str2[j])
{
    if (str1[j] == '\0' || str2[j] == '\0')
    {
        break;
    }
    j++;
}
```

从第 `0` 个字符开始逐个比较。如果当前字符相等，就继续比较下一个字符。如果遇到 `\0`，说明到字符串结尾了。

最后判断：

```c
if (str1[j] == '\0' && str2[j] == '\0')
{
    flag = 1;
}
else
{
    flag = 0;
}
```

如果两个字符串在同一个位置都结束，说明完全相等。否则说明不相等。

### 原题改错点

1. 字符数组定义错误：

```c
char str1(81), str2(81);
```

应改为：

```c
char str1[81], str2[81];
```

2. 输入字符串时不需要加 `&`：

```c
scanf("%s%s", str1, str2);
```

数组名本身就表示首地址。

3. 比较字符时要用 `==`，不能用 `=`：

```c
while (str1[j] == str2[j])
```

4. 相等时应令 `flag = 1`，不相等时令 `flag = 0`。

## E3-5.C：二维数组上下行交换

### 题目功能

定义一个 `N × N` 的二维数组，把：

- 第一行和最后一行交换
- 第二行和倒数第二行交换
- 以此类推

本题中 `N = 5`。

### 数组自动赋值

```c
a[i][j] = rand() % 30;
```

表示随机生成 `0` 到 `29` 的整数。

### 交换思路

```c
i = 0;
j = N - 1;
while (i < j)
```

当 `N = 5` 时：

- 第一次交换第 `0` 行和第 `4` 行
- 第二次交换第 `1` 行和第 `3` 行
- 第 `2` 行是中间行，不需要交换

所以条件是 `i < j`。

### 交换一行中的元素

```c
for (k = 0; k < N; k++)
{
    t = a[i][k];
    a[i][k] = a[j][k];
    a[j][k] = t;
}
```

这是用临时变量 `t` 交换两个元素。因为要交换整行，所以要对这一行的每一列都交换一次。

### 原题填空点

原题空可以填写为：

```c
a[i][j] = rand() % 30;
j = N - 1;
a[i][k] = a[j][k];
j--;
```

## E3-6.C：生成 m × m 乘法二维数组

### 题目功能

输入 `m`，其中 `2 <= m <= 9`，在 `m × m` 的二维数组中存放乘法结果并输出。

输入 `3` 时输出：

```text
123
246
369
```

输入 `5` 时输出：

```text
12345
246810
3691215
48121620
510152025
```

### 数据规律

第 `i` 行第 `j` 列的值是：

```c
(i + 1) * (j + 1)
```

因为数组下标从 `0` 开始，而乘法表从 `1` 开始。

例如 `i = 1, j = 2`：

```text
(i + 1) * (j + 1) = 2 * 3 = 6
```

### 为什么数组定义为 `a[9][9]`

因为题目规定最大 `m` 是 `9`，所以 `9 × 9` 足够存储所有情况。

### 输出格式说明

程序中使用：

```c
printf("%d", a[i][j]);
```

这是为了贴近题目示例中的连续输出。如果老师要求更整齐，也可以使用：

```c
printf("%4d", a[i][j]);
```

## 老师可能会问的问题

### 1. C 语言数组下标为什么从 `0` 开始？

可以回答：C 语言数组第一个元素下标是 `0`，所以长度为 `3` 的数组下标是 `0, 1, 2`，不能访问下标 `3`。

### 2. `flag` 有什么作用？

可以回答：`flag` 是标志变量，用来记录某个条件是否成立。一般 `1` 表示成立，`0` 表示不成立。

### 3. `=` 和 `==` 有什么区别？

可以回答：`=` 是赋值，`==` 是判断是否相等。例如 `flag = 1` 是把 `1` 赋给 `flag`，`flag == 1` 是判断 `flag` 是否等于 `1`。

### 4. 字符串为什么要判断 `\0`？

可以回答：C 语言字符串以 `\0` 作为结束标志，所以判断字符串是否结束要看是否遇到 `\0`。

### 5. 为什么输入字符串时数组名不用加 `&`？

可以回答：数组名本身就表示数组首地址，所以 `scanf("%s", str)` 不需要写成 `&str`。

### 6. 十进制转换进制为什么要倒序输出？

可以回答：除基取余法先得到最低位，后得到最高位，所以存入数组后要倒序输出。

### 7. 二维数组为什么常用两层循环？

可以回答：二维数组有行和列，外层循环通常控制行，内层循环控制列，所以常用两层循环遍历所有元素。

## 复习口诀

- 数组下标从 `0` 开始
- 二维数组：外层行，内层列
- 字符串结尾是 `\0`
- `=` 是赋值，`==` 是比较
- 字符数组输入时数组名不用加 `&`
- 十进制转进制：除基取余，倒序输出
- 交换两个变量：使用临时变量 `t`
